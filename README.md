## Descripcion

- Aplicacion del clima hecha en React con hooks (useContext - useReducer) y maquetado con styled components.

### Dependenciias

- Clonar el proyecto y ejecutar los siguientes comandos

```
 npm install
```

### Ejecucion del programa

```
npm start
```
